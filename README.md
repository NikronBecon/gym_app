# Gym Progress

Локальное iPhone-приложение для планирования тренировок и отслеживания рабочих весов. Backend, аккаунты, облачная синхронизация и сетевые запросы отсутствуют.

## Возможности

- три стартовых шаблона тренировок;
- локальный каталог из 18 упражнений с GIF;
- подходы, веса в кг/фунтах и повторения;
- альтернативные упражнения;
- таймер отдыха;
- календарь и локальные напоминания;
- вес тела, калории и заметка после тренировки;
- графики объёма, рабочих весов и веса тела.

## Запуск

1. Установить полный Xcode 16 или новее.
2. Открыть `GymProgress.xcodeproj`.
3. Выбрать любой симулятор iPhone с iOS 17+.
4. Запустить схему `GymProgress`.

Для установки на физический iPhone нужно выбрать свою Team в Signing & Capabilities. Приложение не использует CloudKit, HealthKit и серверные push-уведомления.

## Хранение данных

История хранится в SwiftData только внутри приложения и исключается из резервной копии iCloud. Удаление приложения удаляет историю. Экспорт и импорт в первой версии не реализованы.

## Медиа

Метаданные упражнений основаны на [hasaneyldrm/exercises-dataset](https://github.com/hasaneyldrm/exercises-dataset). GIF принадлежат Gym Visual и используются с обязательной атрибуцией в разрешении 180×180. Клонирование исходного датасета не предоставляет отдельную лицензию на медиа — перед распространением приложения нужно проверить условия правообладателя.
